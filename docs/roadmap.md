# Roadmap

This document outlines the high-level roadmap and future enhancements planned for CrowdCanvas.

## Phase 1: Foundation & Core AI (Completed)
- Next.js 16 App Router UI/UX
- Supabase Authentication & RLS
- Event Management & Role-Based Access Control
- Media Upload via Cloudinary
- Facial Recognition Enrollment and Matching
- AI Auto-tagging with Google Gemini
- Semantic Search using Transformers.js

## Phase 2: Performance & Scalability (In Progress)
- [ ] Migrate notification infrastructure to real-time WebSockets (Supabase Realtime)
- [ ] Optimize face-api model loading times on the client side via service workers
- [ ] Implement lazy-loaded image grids with progressive blurred placeholders

## Phase 3: Mobile & Native Experience (Planned)
- [ ] Develop a native mobile app using React Native or Capacitor
- [ ] Direct "Share to CrowdCanvas" mobile intent handlers
- [ ] Push Notifications for instant event alerts

## Phase 4: Enterprise & Privacy Features (Planned)
- [ ] End-to-End Encryption (E2EE) for completely private events
- [ ] Granular analytics dashboard for event organizers (demographics, engagement)
- [ ] White-labeling support for organizational event hosting
